Minor Web Developer

CSS to the Rescue

In this course the focus is around building a responsive website/web app for a complex layout. With this we must learn the principles of CSS. The website/web app will be build with the principles of progressive enhancement. This enables designers to design websites according to some predetermined support baseline and then gradually improve and optimize appearance and functionality to accommodate the most advanced browsers.

Exercises

  • Flexbox Baby

Webapp from Scratch

In this course the focus is around building a web application from scratch. The reason why this course exists is to get a better understanding of single page web applications. What happens behind the scenes of Angular for example. The end goal is to make an web application with plain Javascript that can do the basic functionalities like getting data from an API, routing, setting the right template and handle interaction.

Exercises

  • Research the pro's and con's of libaries
  • Research the pro's and con's of webapps
  • Refactor bad code to slightly better code
  • Start the webapp, begin with routing
